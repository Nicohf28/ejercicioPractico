# Ejercicio Práctico
Servidor Express con TypeScript que responde en http://localhost:3000 y muestra el mensaje "Servidor Web en ejecución", a través de un archivo server.ts.

## Clonar y Ejecutar el Proyecto

### 1. Clonar el repositorio:
Para clonar el repositorio en tu máquina local, ejecuta el siguiente comando en la terminal:

- git clone https://github.com/tu_usuario/tu_repositorio.git

### 2. Instalar dependencias:
Una vez clonado el proyecto, navega a la carpeta del proyecot y ejecuta el siguiente comando para instalar dependencias:
- cd ejercicioPractico
- npm install

### 3. Compilar el proyecto:
Compila el código TypeScript a JavaScript con el siguiente comando:
- npx tsc

### 4. Ejecuta el servidor:
Una vez compilado el proyecto, ejecuta el servidor con:
- node dist/server.js

El servidor estará corriendo en http://localhost:3000 y mostrará el mensaje "Servidor Web en ejecución".
